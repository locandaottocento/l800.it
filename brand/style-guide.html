08 · Variabili CSS

Design tokens

Importa il file brand/css/l800-tokens.css nel sito per avere tutti i colori, font e spaziature come variabili CSS.

/* ---- COLORI BRAND ---- */
--l800-tabacco:        #6f3b1c;   /* marrone tabacco principale */
--l800-tabacco-scuro:  #4a2612;   /* tabacco profondo per fondi */
--l800-sabbia:         #d9b988;   /* sabbia principale          */
--l800-sabbia-chiara:  #ecd9b6;   /* crema / fondo              */
--l800-crema:          #fbf7ef;   /* fondo pagina caldo         */
--l800-inchiostro:     #2a1a10;   /* testo principale           */

/* ---- TIPOGRAFIA ---- */
--l800-font-serif:     "Cormorant Garamond", Georgia, serif;
--l800-font-sans:      "Inter", system-ui, sans-serif;

/* ---- TRACKING ---- */

09 · Regole d'uso

Fare · Non fare

Poche regole per mantenere coerenza tra sito, social, stampati e packaging.

✓ Fare
  • Mantenere proporzioni originali del logo
  • Lasciare un margine minimo pari all'altezza del palazzo
  • Usare la versione tabacco/cream su fondi scuri
  • Affiancare al logo solo colori della palette ufficiale
✕ Non fare
  • Modificare i colori del logo
  • Distorcere, ruotare o applicare ombre / gradienti
  • Sostituire il lettering "L'800" con altri font calligrafici
  • Sovrapporre il logo a immagini molto contrastate senza riquadro
